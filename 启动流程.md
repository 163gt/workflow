## 后端：

```bash
# 项目目录下安装pm2
npm install pm2 -g

#启动服务
pm2 start src/index.js

# 4. 查看状态
pm2 ls

# 5. 配置开机自启
pm2 startup
pm2 save

pm2 ls          # 查看服务状态（online=正常）
pm2 logs        # 查看接口日志、报错信息
pm2 restart 0   # 代码更新后，重启服务生效
pm2 stop 0      # 服务关闭
```

前端：
```bash
systemctl restart nginx  # 重启服务
```
