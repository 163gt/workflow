# 欢迎使用工作流系统

这是一个可视化工作流编辑器，帮助您自动化日常任务。

## 主要功能

- **可视化编辑**: 通过拖拽节点来构建工作流
- **多种节点类型**: HTTP 请求、数据处理、条件分支、文件保存等
- **定时执行**: 支持 Cron 表达式设置定时任务
- **执行日志**: 完整的执行记录和状态追踪

## 快速开始

### 1. 创建工作流

1. 点击左侧「+」按钮创建新工作流
2. 从左侧面板拖拽节点到画布
3. 连接节点定义执行流程
4. 点击「保存」按钮保存工作流

### 2. 执行工作流

- **手动执行**: 点击「执行」按钮立即运行
- **定时执行**: 点击「定时」按钮设置执行时间

## 节点说明

| 节点类型 | 功能 |
|---------|------|
| 开始 | 工作流入口点 |
| 结束 | 工作流结束点 |
| HTTP | 发送 HTTP 请求 |
| 数据处理 | 转换和处理数据 |
| 条件 | 根据条件分支执行 |
| 保存文件 | 将数据保存为文件 |
